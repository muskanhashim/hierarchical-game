What is hierarchical clustering?

You have a set of data points: like numbers on a line. No one tells you how many groups exist. The algorithm starts with each point alone, then at every step asks: which two points (or clusters) are currently closest? It merges them, and repeats.

The result is a dendrogram, a tree where the height of each branch encodes how far apart two things were when they joined. Low branch = very similar. High branch = forced together as a last resort. You can then "cut" the tree at any height to decide how many clusters you want. This is the key decision in the method.
